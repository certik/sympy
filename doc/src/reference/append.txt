Append
======

http://reference.wolfram.com/mathematica/ref/Append.html

expr + [elem]
    gives expr with elem appended

expr.append(elem)
    appends elem to expr inplace

Examples
--------

Basic Examples
~~~~~~~~~~~~~~

::
    In [1]: var("a b c d")
    Out[1]: (a, b, c, d)

    In [2]: [a, b, c, d] + [x]
    Out[2]: [a, b, c, d, x]
